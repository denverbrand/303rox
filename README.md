# 303ROX®

Denver's independent HYROX hub. Race intel, gyms, run clubs, training, gear.

🌐 **Live site:** [303rox.com](https://303rox.com)

## What's in this repo

- `index.html` — the entire site (self-contained, no build step)
- `303rox-logo.png` — the wordmark logo

## How updates work

This repo is connected to SiteGround via Git deploy. Push to `main` and the live site updates automatically within ~30 seconds.

```bash
# Make changes to index.html
git add .
git commit -m "describe your change"
git push origin main
# Done — live in 30 seconds
```
